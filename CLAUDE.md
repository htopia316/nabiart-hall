# 나비아트홀 (Nabiart Hall) - 극단 공식 홈페이지

## 프로젝트 개요
나비아트홀 극단의 공식 홈페이지. 공연 정보, 티켓 예매, 극장 대관, 후원 등 극단 운영에 필요한 모든 기능을 제공하는 풀스택 웹 애플리케이션.

## 기술 스택
- **프론트엔드**: Next.js 14+ (App Router), TypeScript, Tailwind CSS
- **상태관리**: Zustand
- **다국어**: next-intl (한국어 기본, 영어, 중국어)
- **백엔드/DB**: Supabase (Auth, Database, Storage, Realtime)
- **결제**: 토스페이먼츠 (티켓 예매, 후원)
- **배포**: Vercel
- **폰트**: Pretendard (한국어/영어), PingFang SC 또는 Noto Sans SC (중국어)

## 디자인 원칙
- Apple/Toss 스타일: 미니멀, 직관적, 깔끔한 UI/UX
- 반응형: PC (1440px+) / 태블릿 (768px~1439px) / 모바일 (~767px)
- 테마: 시스템 테마(기본), 라이트 모드, 다크 모드
- 애니메이션: 부드러운 트랜지션, Framer Motion 활용
- 접근성: WAI-ARIA 준수, 키보드 네비게이션

## 폴더 구조 (예상)
```
src/
├── app/
│   ├── [locale]/           # 다국어 라우팅
│   │   ├── layout.tsx
│   │   ├── page.tsx        # 랜딩 페이지
│   │   ├── shows/          # 공연
│   │   ├── booking/        # 예매
│   │   ├── people/         # 배우/스텝
│   │   ├── rental/         # 대관
│   │   ├── notices/        # 공지사항
│   │   ├── support/        # 후원
│   │   └── admin/          # 관리자
│   └── api/                # API Routes
├── components/
│   ├── ui/                 # 공통 UI 컴포넌트
│   ├── layout/             # 레이아웃 컴포넌트
│   └── features/           # 기능별 컴포넌트
├── lib/
│   ├── supabase/           # Supabase 클라이언트
│   ├── i18n/               # 다국어 설정
│   └── utils/              # 유틸리티
├── messages/               # 번역 파일
│   ├── ko.json
│   ├── en.json
│   └── zh.json
├── styles/                 # 글로벌 스타일
└── types/                  # TypeScript 타입 정의
```

## 컨벤션
- 컴포넌트: PascalCase (예: `ShowCard.tsx`)
- 유틸/훅: camelCase (예: `useBooking.ts`)
- 커밋 메시지: Conventional Commits (feat:, fix:, chore: 등)
- 브랜치: feature/T-XXX-description 형식
